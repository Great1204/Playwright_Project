# Playwright Automation Project

## Project Description
This project contains automated test scripts using Playwright for testing the Automation Exercise website.

## Tech Stack
- Playwright (JavaScript)
- Node.js

## Project Structure
- tests/ → Test cases
- pages/ → Page Object Model
- utils/ → Utility files
- playwright.config.js → Configuration

## How to Run Tests
</> bash
npx playwright test

## Run in Headed Mode
npx playwright test --headed

## Report
npx playwright show-report
